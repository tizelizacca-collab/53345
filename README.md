# Analizador Sintáctico con ANTLR4

Proyecto realizado para la materia SySL utilizando ANTLR4 y JavaScript.

## Descripción

El proyecto implementa un analizador léxico, sintáctico e intérprete simple para un lenguaje que permite:

- estructuras condicionales (`si`, `sino`, `entonces`)
- impresión de mensajes (`imprimir`)
- condiciones booleanas (`verdadero`, `falso`)
- instrucción de salida (`salir`)

El analizador fue desarrollado utilizando:

- ANTLR4
- Node.js
- JavaScript

---

# Requisitos

Para ejecutar el proyecto se necesita:

- Node.js
- Java
- ANTLR 4
